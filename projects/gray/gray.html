Overview
I live in a bubble of 0 and 1s, yet I believe that the world around me is not binary. There is no absolute right or wrong, no absolute gender, no absolute religion, no absolute race, no absolute sexuality, no absolute black or white - but gray.
This is my latest project - Gray, an ode to the struggles of unique individuals in our perceived binary world.Â

Implementation
This is a fairly simple implementation using SVGs and D3. You can use vanilla JavaScript and Canvas as well - it would not make a difference. The interesting part about this project is that I had to implement the physics myself for the balls. Writing code for Elastic Collisions in 2-Dimensions was extremly fun - I had to revise some high school physics and math.
